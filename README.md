# KeenKeeper

A beautiful friendship management app to help you stay connected with the people who matter most.

## Description

FriendTrack helps you track your relationships, set contact goals, and log interactions so you never lose touch with friends. Built with a modern Ocean Deep design theme featuring Sora and Manrope typography.

## Technologies Used

- **React.js** — Component-based UI framework
- **React Router DOM** — Client-side routing
- **Tailwind CSS** — Utility-first CSS styling
- **Recharts** — Interactive pie charts for analytics
- **Lucide React** — Beautiful icon set
- **Next.js** — React framework for building server-rendered applications and static websites

## Key Features

1. **Friend Dashboard** — View all friends at a glance with status indicators (overdue, almost due, on-track) and quick access to detailed profiles
2. **Quick Check-In** — Log calls, texts, and video interactions with one click, automatically building your relationship timeline
3. **Friendship Analytics** — Visualize your interaction patterns with a pie chart breakdown of calls, texts, and video sessions
